02 — Getting Started

Quick Start

A minimal page using Silicon Zen. All classes are prefixed sz- so they never clash with your existing CSS.

<!DOCTYPE html> <html lang="en" data-sz-theme="parchment"> <head> <link rel="stylesheet" href="sz.css"> <!-- Google Fonts --> </head> <body class="sz-body sz-dither"> <div class="sz-container"> <span class="sz-label">01 — Intro</span> <h1 class="sz-display">High-Fidelity Nostalgia.</h1> <p class="sz-lead">Silicon Zen UI.</p> <a href="#" class="sz-btn sz-btn--solid">[ Start ]</a> </div> <script src="sz.js"></script> </body></html>
03 — Foundations

Themes

Two themes ship out of the box: Parchment (light, serif-calm) and Obsidian (dark, terminal-dense). Apply to any element.

<!-- Apply on <html> for full-page theming --> <html data-sz-theme="parchment"> <html data-sz-theme="obsidian"> <!-- Or scope to any element --> <section data-sz-theme="obsidian">...</section> <!-- Toggle button (wired automatically by sz.js) --> <button data-sz-theme-toggle>Toggle</button> <!-- Set specific theme on click --> <button data-sz-theme-set="obsidian">Go Dark</button> /* JS API */ SZ.theme('obsidian'); SZ.toggleTheme();
04 — Foundations

Design Tokens

All values are CSS custom properties under --sz-*. Override in :root to reskin without touching any component.

/* Override any token in your own CSS */ :root { /* Accents — shared across both themes */ --sz-teal: #5BA8A0; --sz-teal-dim: #3D7A74; --sz-amber: #FFB86C; --sz-amber-dim: #CC8B3D; --sz-red: #D94F3D; --sz-green: #4FA878; /* Surface — Parchment defaults */ --sz-bg: #F4F4F1; --sz-bg-alt: #EAEAE7; --sz-surface: #FFFFFF; --sz-surface-alt: #F0F0ED; --sz-border: #333333; --sz-border-dim: #888888; --sz-text: #0D0D0D; --sz-text-dim: #555555; /* Typography */ --sz-font-serif: "Newsreader", Georgia, serif; --sz-font-mono: "JetBrains Mono", monospace; --sz-font-pixel: "Press Start 2P", monospace; /* Grid */ --sz-grid: 4px; /* all spacing = multiples of this */ /* Motion */ --sz-snap: 150ms cubic-bezier(0, 0, 0.2, 1); --sz-spring: 200ms cubic-bezier(0.34, 1.56, 0.64, 1); }

14 — Components

Accordion

A retro-pixel + modern-minimalist design language and component library by king525dev. It treats the pixel as a deliberate unit of construction for the calm internet.
Three layers: Newsreader (serif, content), JetBrains Mono (monospace, UI/data), and Press Start 2P (pixel, brand moments only).
sz.css is pure CSS — no JS required. sz.js adds interactivity for modals, tabs, accordions, toast, and theme toggling. It's optional.
<!-- data-sz-single: only one item open at a time --> <div class="sz-accordion" data-sz-single> <div class="sz-accordion__item is-open"> <button class="sz-accordion__trigger">Question?</button> <div class="sz-accordion__body">Answer text.</div> </div> <div class="sz-accordion__item"> <button class="sz-accordion__trigger">Another?</button> <div class="sz-accordion__body">Answer.</div> </div> </div>
