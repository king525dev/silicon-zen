Common Commands

sz build --out ./dist --watch

Parses all templates, compiles 8 modules, optimises images. Outputs to ./dist. Use --watch for live reloading.

sz optimize --aggressive --format webp

Runs dithering pass, converts images to WebP, strips unused CSS. Average reduction: 66%.

sz deploy --env prod --confirm

Runs build, optimise, then pushes to the configured host. Requires --confirm flag in production.

sz theme obsidian

Switches the active design theme. Tokens compile to CSS custom properties. Both themes hot-reload without a page refresh.
